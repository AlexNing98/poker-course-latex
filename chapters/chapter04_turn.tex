\chapter{转牌 / Turn}

\section{本章导入}

转牌是整手牌中策略变化最明显的一条街。翻牌发出三张公共牌后，双方已经完成了第一轮范围碰撞；到了转牌，第四张公共牌出现，牌面结构、双方牌力、听牌权益和下注压力都会发生新的变化。

很多新手在转牌圈最大的问题，是把翻牌决策机械延续到转牌。翻牌跟注了，转牌就觉得“都跟到这里了，再看一张”；翻牌下注了，转牌就觉得“我已经代表强牌，必须继续打”；翻牌拿到顶对，转牌即使出现明显危险牌，也不愿意承认牌力已经下降。

但转牌不是翻牌的简单重复。转牌是一条重新评估的街道。翻牌时很多牌力还处于模糊状态，例如顶对可能仍然领先，听牌仍然有两张牌可以追，中等牌力仍然有摊牌希望；但到了转牌，公共牌增加，底池变大，剩余可发牌减少，下注成本提高，每一个继续决策都变得更加昂贵。

因此，转牌圈的核心不是“我翻牌怎么打，转牌就继续怎么打”，而是要重新问：这张转牌改变了什么？它强化了谁的范围？它提升了我的具体牌力，还是削弱了我的牌力？我的下注是否仍然是价值或诈唬？如果我面对下注，继续跟注是否仍然满足条件？

本章的目标，是帮助新手建立转牌圈的基础判断框架。你需要理解为什么转牌是“转变”的街道，什么是好转牌和坏转牌，进攻方什么时候适合继续开火，防守方什么时候应该继续、加注或弃牌，听牌在转牌上的价值为什么会下降，以及大额下注为什么往往意味着更极化的范围。

本章的核心输出是“转牌三类出张”：

\begin{center}
\textbf{强化我范围的牌：可以继续进攻。\\
强化对手范围的牌：需要谨慎。\\
空白牌：延续翻牌逻辑，但重新评估下注目的。}
\end{center}

只要你能在转牌圈先判断这张牌属于哪一类，就能避免大量“翻牌跟了，转牌无脑继续”的错误。

\section{核心概念}

\subsection{为什么转牌是“转变”的街道}

转牌之所以重要，是因为它同时带来了四种变化。

第一，牌面结构发生变化。翻牌上的干燥牌面，可能因为转牌出现同花听牌或顺子听牌而变得复杂；翻牌上的湿润牌面，可能因为转牌完成同花或顺子而让某些听牌变成成牌；翻牌上的单对牌面，也可能因为转牌配对而让暗三条变成葫芦，或者让两对价值发生变化。

第二，双方范围重新分化。翻牌下注和跟注之后，双方范围都已经变窄。进攻方继续下注后，通常保留了更强的价值牌和部分诈唬；防守方跟注后，也过滤掉了大量完全没有权益的空气牌。因此到了转牌，双方范围比翻牌更集中，继续决策也更需要精确。

第三，听牌权益下降。翻牌时听牌还有两张牌可以看，转牌时只剩一张河牌。很多听牌在翻牌可以合理继续，但到了转牌面对大注时，继续成本大幅提高，而完成概率下降。如果没有足够赔率、隐含赔率或弃牌权益，转牌继续追听牌往往会变成亏损。

第四，下注压力增加。翻牌底池较小，下注成本相对低；转牌底池已经变大，下注金额通常更高。一个转牌下注，往往会为河牌大底池甚至全下做准备。因此，转牌下注比翻牌下注更能表达强范围，也更能给防守方制造压力。

正因为这四种变化，转牌不是简单的“第二次翻牌”。它是整手牌从初步试探走向明确分化的关键节点。

\subsection{转牌出张如何改变牌力}

同一手牌在翻牌和转牌上的价值可能完全不同。转牌出张会通过几个方式改变牌力。

第一，直接提升你的手牌。例如你翻牌持有同花听牌，转牌完成同花；你翻牌持有两头顺听牌，转牌完成顺子；你翻牌持有顶对，转牌击中第二对成为两对。这类转牌显著提升你的具体牌力，通常可以支持更积极的行动。

第二，直接提升对手的可能范围。例如翻牌为 \texttt{KJ7 two-tone}，转牌完成同花。即使你仍然持有顶对，对手翻牌跟注范围中的大量同花听牌已经完成。你的牌力相对下降，需要谨慎。

第三，改变牌面威胁。例如翻牌为 \texttt{A72 rainbow}，转牌为 \texttt{T}，整体仍然较干燥；但如果转牌为第二张同花，牌面开始出现同花听牌，对手继续范围可能增加更多权益。转牌不一定立即完成强牌，但可能增加未来河牌压力。

第四，压缩或扩大价值下注范围。有些转牌让强牌更强、弱牌更弱，适合极化下注；有些转牌让双方范围都获得中等权益，不适合盲目大额下注。

判断转牌时，不能只看“这张牌有没有帮到我”，还要看“这张牌是否更可能帮到对手”。扑克决策永远是双方范围之间的比较，而不是你和公共牌之间的比较。

\subsection{好转牌与坏转牌}

所谓好转牌，不是指绝对意义上的好牌，而是指相对于你的范围和具体手牌，它更有利于你继续执行策略。所谓坏转牌，则是指它更可能强化对手范围，削弱你的价值，或让你后续难以继续。

例如，你在前位 open，大盲跟注，翻牌为 \texttt{A72 rainbow}，你小注持续下注，大盲跟注。转牌如果是 \texttt{K}，这通常更有利于前位加注者。因为你的范围中有更多 \texttt{AK}、\texttt{KK}、\texttt{AQ} 等强高牌组合。即使你持有部分诈唬牌，也可以利用这张 K 继续施压。

但如果翻牌为 \texttt{987 two-tone}，你作为前位加注者下注，大盲跟注，转牌为 \texttt{6} 或 \texttt{T}，这类牌明显强化了大盲防守范围中的顺子和两对组合。即使你持有 \texttt{AA}，也必须谨慎，因为对手范围已经获得大量强牌。

好转牌通常具有以下特点：

\begin{itemize}
\item 强化你的翻前强范围；
\item 提升你的具体牌力；
\item 让对手大量中等牌力承受压力；
\item 没有明显完成对手的主要听牌；
\item 支持你继续讲述一致的强牌故事。
\end{itemize}

坏转牌通常具有以下特点：

\begin{itemize}
\item 明显完成对手翻牌跟注范围中的听牌；
\item 更容易击中对手防守范围中的连接牌；
\item 削弱你单对牌的相对价值；
\item 让你继续下注后很难面对加注；
\item 让你的翻牌诈唬失去可信度。
\end{itemize}

空白转牌则是没有明显改变局势的牌。例如翻牌为 \texttt{K72 rainbow}，转牌为 \texttt{3}，大多数情况下并未显著改变双方范围。此时可以延续翻牌逻辑，但仍然需要重新评估下注目的。

\subsection{进攻方如何选择继续开火}

进攻方通常指翻前加注并在翻牌下注的一方。到了转牌，进攻方是否继续开火，需要综合考虑转牌出张、自己的范围优势、具体手牌类别、对手翻牌跟注范围以及河牌计划。

继续开火的常见理由包括：

\begin{itemize}
\item 转牌强化了你的范围；
\item 你的具体手牌仍然可以从更差牌中获得价值；
\item 你持有强听牌或合适阻断牌，适合继续诈唬；
\item 对手翻牌跟注范围中有大量中等牌力，会在转牌面对压力；
\item 你的下注可以为河牌全下或大额价值下注铺路。
\end{itemize}

例如你翻前 open，翻牌在 \texttt{A72 rainbow} 上 C-bet，被大盲跟注。转牌为 \texttt{K}。这张 K 对你的范围非常有利，你可以用 \texttt{AK}、\texttt{AA}、\texttt{KK}、强 A 继续价值下注，也可以用部分带有 Q、J、T 的牌继续施压，因为对手的许多 7x、小对子和弱 A 会承受压力。

但并不是所有翻牌下注后都应该转牌继续。以下情况通常需要放慢：

\begin{itemize}
\item 转牌明显强化对手范围；
\item 你的牌属于中等摊牌价值，下注会让更差牌弃掉、更强牌继续；
\item 对手翻牌跟注范围非常强；
\item 你的诈唬牌没有足够权益或阻断效果；
\item 被加注时你无法继续。
\end{itemize}

新手常见错误是“开了一枪就想开第二枪”。但转牌第二枪比翻牌第一枪贵得多，也更能暴露策略问题。每一枪都必须重新确认：这张牌是否支持我继续讲这个故事？

\subsection{防守方如何重新评估跟注范围}

防守方在翻牌跟注后，到了转牌不能机械继续。翻牌跟注只说明你在翻牌价格下可以继续，并不代表转牌面对任何下注都必须继续。

防守方转牌决策至少要考虑五点。

第一，转牌是否改善了我的牌力。如果你从听牌变成成牌，或者从一对变成两对，可以考虑继续甚至反击；如果转牌没有改善，且对手继续大额下注，你需要重新计算继续条件。

第二，转牌是否改善了对手范围。有些转牌虽然没有直接击中对手具体手牌，但明显有利于进攻方整体范围。面对这类转牌的第二枪，你不能只看自己还有一对，而要判断对手价值范围是否变强。

第三，我的牌在防守范围中处于什么位置。如果你持有的是防守范围中较强的成牌或强听牌，可以继续；如果只是底对、弱中对或低质量听牌，面对转牌下注经常应该弃牌。

第四，我是否有足够赔率。转牌只剩一张河牌，听牌完成概率下降。如果对手下注较大，你继续追听牌所需的胜率会更高。

第五，我是否有反击空间。有些牌不适合单纯跟注，但可以在特定牌面作为加注半诈唬或价值加注候选。不过对于初学者来说，转牌加注需要谨慎，因为它通常代表更强、更极化的范围。

防守方必须理解：翻牌跟注是一项独立决策，转牌跟注也是一项新的独立决策。已经投入的筹码属于底池，不应该成为继续犯错的理由。

\subsection{听牌在转牌的价值变化}

听牌在翻牌和转牌上的价值差别很大。翻牌时，你还有两张牌可以改善；转牌时，只剩一张河牌。这个变化会显著影响听牌是否值得继续。

例如，你在翻牌持有同花听牌，面对小注跟注可能合理。因为你有两次机会完成同花，也可能在后续街获得额外价值。但到了转牌，如果同花没有完成，对手再次大额下注，你只剩一张牌可以追。此时如果下注尺度很大，而你没有其他权益或隐含赔率，继续跟注可能就不再合理。

听牌在转牌的处理通常分为三类。

第一类是强听牌。比如高同花听牌加两张高牌、同花听牌加顺子听牌、开放式顺子听牌加一对。这类牌即使未完成，也可能具有较高权益，甚至适合在某些牌面上作为半诈唬继续进攻。

第二类是普通听牌。比如单纯同花听牌或两头顺听牌。它们是否继续，要看下注价格、位置、对手类型和隐含赔率。

第三类是弱听牌。比如低质量卡顺、低同花听牌、可能被更高同花压制的听牌。这类牌面对转牌大注时，往往应该弃掉。

新手最需要避免的是“我都追到转牌了，再看一张”。这句话是典型的沉没成本思维。转牌继续与否，取决于当前价格和未来收益，而不是你前面已经投入了多少。

\subsection{中等牌力为什么经常需要控池}

中等牌力在转牌上尤其容易出问题。翻牌时，中等牌力可能还比较舒服，比如顶对一般踢脚、第二对子、口袋中对子等。但到了转牌，如果底池变大、对手继续下注、牌面出现危险变化，这类牌的处境会迅速恶化。

中等牌力的核心特点是：它能击败一些诈唬和较弱牌，但通常不能承受多条街大额下注。它的主要目标不是无限扩大底池，而是控制底池并实现摊牌价值。

例如你持有 \texttt{KTs}，翻牌为 \texttt{KJ7 two-tone}，你下注被大盲跟注。转牌为 \texttt{Q}。这张牌让 \texttt{AT} 成顺，也让 \texttt{AQ}、\texttt{KQ}、\texttt{QT}、\texttt{QJ} 等牌力提升，同时许多听牌仍然存在。你的顶对已经不再像翻牌那样舒服。如果继续大额下注，可能让更差牌弃掉，让更强牌继续。此时控池往往比盲目开第二枪更合理。

控池不是放弃，而是让底池大小匹配牌力。中等牌力可以通过过牌、跟注小注、控制下注频率等方式实现摊牌价值。新手要避免把中等牌力打成强价值牌，因为一旦底池膨胀，中等牌力通常会成为被价值下注的对象。

\subsection{大额下注与极化范围}

转牌的大额下注通常比翻牌更有压力。原因很简单：底池已经变大，下注金额更高，而且转牌大注往往会为河牌全下或大额下注做准备。

大额下注通常对应更极化的范围。所谓极化范围，是指下注者主要用两类牌下注：非常强的价值牌和没有足够摊牌价值但适合诈唬的牌。中等牌力通常不适合大额下注，因为它们不希望被更差牌大量弃牌，也不希望面对加注或大底池。

例如在 \texttt{A72K} 这样的牌面上，进攻方可能用 \texttt{AK}、\texttt{AA}、\texttt{KK} 等强牌大额下注，也可能用部分没有摊牌价值但带阻断效果的牌诈唬。防守方的 7x、小对子、弱 A 会承受很大压力。

使用大额下注需要满足几个条件：

\begin{itemize}
\item 你拥有足够强的价值牌支撑；
\item 这张转牌有利于你的范围或坚果优势；
\item 对手范围中有较多中等牌力会被施压；
\item 你的诈唬候选具有一定阻断效果或后续权益；
\item 你知道河牌如何继续。
\end{itemize}

新手不应为了“显得强”而随便使用大额下注。大额下注不是情绪表达，而是范围结构和底池规划的结果。

\section{新手常见误区}

\subsection{误区一：翻牌跟了，转牌无脑继续跟}

这是娱乐玩家最常见、也是最昂贵的转牌错误之一。很多人翻牌跟注后，转牌面对第二枪时会想：“我已经跟了一枪，现在弃牌太可惜了。”这种想法本质上是沉没成本思维。

已经投入底池的筹码不再属于你。你不能因为前面投过钱，就继续用错误价格追牌或抓诈唬。转牌是否继续，必须重新根据当前牌面、对手范围、下注尺度和你的牌力判断。

如果转牌明显强化对手范围，而你的牌没有改善，面对大额下注时弃牌往往是正确选择。

\subsection{误区二：只要听牌还没死，就继续追}

很多新手会说：“我还有同花听牌。”“我还有顺子机会。”“河牌中了就能赢大底池。”但问题是，听牌是否能继续，取决于价格是否合适。

转牌只剩一张牌，普通听牌完成概率有限。面对大注，如果没有足够直接赔率或隐含赔率，继续追听牌会长期亏损。更糟糕的是，有些听牌即使完成，也可能不是坚果牌。例如低同花听牌完成后，仍可能输给更高同花。

有机会不等于可以跟。扑克中几乎任何牌都有一点机会，但只有价格合理的机会才值得购买。

\subsection{误区三：转牌危险牌仍然强行价值下注}

一些新手在翻牌拿到顶对或超对后，即使转牌出现明显危险牌，也不愿意放慢。他们会觉得：“我翻牌领先，现在应该继续保护。”结果经常在对手已经完成强牌时继续投入筹码。

例如你持有 \texttt{AA}，翻牌为 \texttt{T98 two-tone}，转牌为 \texttt{7}。这张牌让大量 \texttt{Jx}、\texttt{6x}、两对、顺子相关组合变强。此时 \texttt{AA} 的相对价值已经明显下降。如果你仍然把它当作绝对强牌大额下注，就可能在错误范围上投入过多。

价值下注必须建立在“更差牌会跟注”的基础上。如果转牌让对手范围中大量更强牌出现，你需要先重新评估，而不是机械延续翻牌价值下注。

\subsection{误区四：把第二枪当成必须完成的任务}

有些玩家认为，翻牌 C-bet 后，如果转牌不继续下注，就显得自己很弱。于是他们为了保持强势，几乎所有转牌都开第二枪。

这类打法的问题是，下注没有根据。好的第二枪需要转牌支持你的范围、你的牌适合下注、对手有足够可弃牌，或者你能从更差牌中获得价值。如果这些条件都不存在，继续下注只是在扩大亏损。

强牌手不是每一条街都进攻，而是在适合进攻的时候进攻，在应该放慢的时候放慢。

\subsection{误区五：中等牌力不懂控池}

中等牌力的最大敌人，是牌手自己的贪心和恐惧。贪心让你想多拿价值，恐惧让你怕被反超，于是你不断下注，最终把底池做得过大。

例如你持有弱顶对，在转牌面对复杂牌面时继续大额下注。更差牌可能弃掉，更强牌继续，听牌也可能有足够权益跟注甚至加注。最终你会发现自己在河牌面对大底池，手里却只有一手不适合摊牌大额筹码的中等牌。

控池是一种主动选择。它不是害怕，而是承认自己的牌力适合中小底池，不适合大底池。

\subsection{误区六：忽略河牌计划}

转牌下注之后，通常会直接影响河牌底池大小。如果你转牌下注很大，河牌剩余筹码可能已经适合全下。因此，转牌下注前必须考虑河牌计划。

你需要问：

\begin{itemize}
\item 如果对手跟注，河牌我还会继续价值下注吗？
\item 哪些河牌我会诈唬？
\item 哪些河牌我会放弃？
\item 如果河牌完成明显听牌，我如何处理？
\item 当前转牌下注是否让河牌变得容易决策？
\end{itemize}

没有河牌计划的转牌下注，往往会把自己带进更大的困难。

\section{实战思考框架}

本章的核心框架是“转牌三类出张”：

\begin{center}
\textbf{强化我范围的牌：可以继续进攻。\\
强化对手范围的牌：需要谨慎。\\
空白牌：延续翻牌逻辑，但重新评估下注目的。}
\end{center}

\subsection{第一类：强化我范围的牌}

强化我范围的牌，是指这张转牌更符合我前面行动所代表的范围，也更容易让我拥有强价值牌。

例如你在前位 open，大盲跟注。翻牌为 \texttt{A72 rainbow}，你下注，大盲跟注。转牌为 \texttt{K}。这张牌通常更强化你的范围，因为你有大量 \texttt{AK}、\texttt{KK}、强 A 和高牌组合。大盲虽然也可能有一些 Kx，但整体来看，这张 K 更支持你继续进攻。

面对强化自己范围的转牌，你可以考虑：

\begin{itemize}
\item 用强价值牌继续下注；
\item 用能阻断对手强牌的诈唬继续施压；
\item 对对手中等牌力施加压力；
\item 根据河牌计划选择合适尺度。
\end{itemize}

但“可以继续进攻”不等于“必须用所有牌继续”。你仍然需要区分价值牌、中等牌力、强听牌和空气牌。范围有利只是提供进攻基础，具体行动仍要看手牌功能。

\subsection{第二类：强化对手范围的牌}

强化对手范围的牌，是指这张转牌更可能击中对手翻牌跟注范围，或者完成对手主要听牌。

例如你在前位 open，大盲跟注。翻牌为 \texttt{987 two-tone}，你下注，大盲跟注。转牌为 \texttt{6}。这张牌显著强化大盲范围，因为大盲可以拥有更多 \texttt{T8}、\texttt{85}、\texttt{65}、两对、顺子和组合听牌。你即使持有 \texttt{AA}，也必须谨慎。

面对强化对手范围的转牌，你需要：

\begin{itemize}
\item 降低自动继续下注频率；
\item 用中等牌力更多控池；
\item 避免用没有权益的空气牌盲目开第二枪；
\item 面对对手加注时更加谨慎；
\item 重新评估自己的价值下注是否仍然成立。
\end{itemize}

强化对手范围的转牌，是新手最容易亏大钱的地方。因为他们常常只记得自己翻牌很强，却没有意识到转牌已经改变了局势。

\subsection{第三类：空白牌}

空白牌是指这张转牌没有明显完成听牌，也没有明显改变双方范围结构。

例如翻牌为 \texttt{K72 rainbow}，转牌为 \texttt{3}。这张 3 通常不会显著改变双方范围。进攻方可以根据翻牌逻辑继续价值下注或诈唬；防守方也可以根据自己的牌力继续判断。

但空白牌并不意味着可以不思考。到了转牌，双方范围已经因为翻牌行动变窄，底池也变大。即使是空白牌，也必须重新评估下注目的。

面对空白转牌，你可以问：

\begin{itemize}
\item 我的价值牌是否仍然能被更差牌跟注？
\item 我的诈唬是否仍然能让更好牌弃牌？
\item 对手翻牌跟注后，范围中还有多少弱牌？
\item 我的中等牌力是否适合继续扩大底池？
\item 这次下注会如何影响河牌计划？
\end{itemize}

空白牌通常允许你延续翻牌逻辑，但不允许你停止思考。

\subsection{转牌下注前的五个检查问题}

无论转牌属于哪一类，下注前都建议检查五个问题：

\begin{enumerate}
\item 这张转牌强化了谁的范围？
\item 我的具体手牌属于价值牌、中等牌力、听牌还是空气牌？
\item 我下注是为了价值，还是为了诈唬？
\item 如果被跟注，哪些河牌我会继续？
\item 如果被加注，我是否有清晰应对方案？
\end{enumerate}

如果这些问题回答不清，说明这次下注可能只是机械延续或情绪驱动。此时过牌、控池或放弃，往往比盲目继续更好。

\subsection{转牌面对下注的五个检查问题}

当你作为防守方面对转牌下注时，可以使用另一组检查问题：

\begin{enumerate}
\item 这张转牌是否改善了我的牌？
\item 这张转牌是否明显改善了对手范围？
\item 我的牌在防守范围中是否足够强？
\item 面对当前下注尺度，我是否有足够胜率或隐含赔率？
\item 如果我跟注，河牌面对继续下注是否有计划？
\end{enumerate}

如果你只是因为“翻牌已经跟了”而继续，通常就是错误信号。

\section{牌例拆解}

\subsection{牌例一：强化进攻方范围的好转牌}

\subsubsection*{牌局背景}

有效筹码 100BB。你在中位拿到 \texttt{AQo}，open raise 到 2.5BB。只有大盲跟注。

翻牌为 \texttt{A72 rainbow}。大盲过牌，你下注 1/3 底池，大盲跟注。

转牌为 \texttt{K}。大盲再次过牌，行动轮到你。

\subsubsection*{新手常见想法}

新手可能会想：

\begin{itemize}
\item “我还是顶对，可以继续打。”
\item “转牌来了 K，有点吓人。”
\item “如果他有 K，会不会反超？”
\end{itemize}

这些想法只关注具体手牌，没有充分考虑范围变化。

\subsubsection*{理性分析}

第一，判断转牌类型。\texttt{K} 是强化进攻方范围的牌。你作为中位 open，范围中有大量 \texttt{AK}、\texttt{AQ}、\texttt{KK}、强 A 和高牌组合。大盲虽然也有部分 Kx，但他在翻牌 \texttt{A72} 跟注后，范围中更多是 Ax、7x、小对子和一些浮动牌。

第二，判断具体牌力。\texttt{AQ} 在 \texttt{A72K} 上仍然是较强的顶对。它落后于 \texttt{AK}、\texttt{AA}、\texttt{KK}、\texttt{77}、\texttt{22} 等强牌，但仍然领先许多弱 A、7x 和部分继续牌。

第三，明确下注目的。你的下注主要是价值下注，可以向较弱 A 收取价值，也可以给部分带有权益的牌施压。由于转牌 K 强化你的整体范围，你也可以用一些诈唬牌继续开火，因此你的价值下注有范围支持。

第四，规划河牌。许多空白河牌可以考虑继续薄价值或根据对手类型过牌摊牌。如果河牌出现明显完成听牌或对手主动下注，需要重新评估。

\subsubsection*{推荐行动}

继续下注，使用中等尺度较为合理。

不需要过度害怕 K，因为这张牌整体更有利于你的范围。你的 \texttt{AQ} 仍然可以从较弱 A 中获取价值。

\subsubsection*{本手牌教训}

转牌是否可怕，不能只看它是不是高牌，而要看它强化谁的范围。在这个例子中，K 虽然改变了牌面，但更符合翻前加注方的强范围，因此进攻方可以继续施压。

\subsection{牌例二：强化防守方范围的坏转牌}

\subsubsection*{牌局背景}

有效筹码 100BB。你在前位拿到 \texttt{AA}，open raise 到 3BB。大盲跟注。

翻牌为 \texttt{987 two-tone}。大盲过牌，你下注 2/3 底池，大盲跟注。

转牌为 \texttt{6}。大盲过牌，行动轮到你。

\subsubsection*{新手常见想法}

很多新手会想：

\begin{itemize}
\item “我有 AA，不能给他免费看河牌。”
\item “牌面更危险了，要继续保护。”
\item “如果他有听牌，我要让他付费。”
\end{itemize}

这些想法忽略了一个关键点：转牌 \texttt{6} 已经明显强化大盲范围。

\subsubsection*{理性分析}

第一，判断转牌类型。\texttt{6} 是强化防守方范围的牌。大盲防守范围中有更多 \texttt{T8}、\texttt{85}、\texttt{65}、\texttt{76}、\texttt{98}、\texttt{87}、暗三条和两对组合。翻牌跟注后，这些组合大量保留到转牌。

第二，判断具体牌力。\texttt{AA} 仍然是超对，但在 \texttt{9876} 这个牌面上，它的相对价值已经显著下降。它领先部分一对和听牌，但落后于大量顺子、两对和暗三条。

第三，明确下注目的。如果继续下注，哪些更差牌会跟注？可能有一些听牌和一对加听牌。但如果被 check-raise，你会非常困难。更重要的是，下注可能把自己置于一个被对手强范围惩罚的局面。

第四，规划河牌。任何 \texttt{T}、\texttt{5}、同花完成牌，甚至部分高张，都可能让河牌更加复杂。继续膨胀底池会让后续决策更痛苦。

\subsubsection*{推荐行动}

倾向于过牌控池。

这并不意味着 \texttt{AA} 一定落后，而是因为转牌明显更有利于对手范围，且你的牌不适合在这个结构上继续打大底池。过牌可以控制底池，保留摊牌价值，并避免面对大额加注。

\subsubsection*{本手牌教训}

强牌也会因为转牌出张而降级。翻牌的超对，到了转牌可能只是脆弱的单对。坏转牌出现时，保护不是无脑下注，控池才是更成熟的选择。

\subsection{牌例三：听牌在转牌面对大注}

\subsubsection*{牌局背景}

有效筹码 100BB。你在按钮位拿到 \texttt{QJs}，前面玩家弃牌，你 open raise 到 2.5BB。大盲跟注。

翻牌为 \texttt{A72}，其中两张与你同花。大盲过牌，你下注 1/3 底池，大盲跟注。

转牌为 \texttt{4}，没有完成同花。大盲突然下注接近底池。

\subsubsection*{新手常见想法}

很多新手会想：

\begin{itemize}
\item “我还有同花听牌，不能弃。”
\item “河牌中了就能赢大底池。”
\item “我都已经打到转牌了，再看一张。”
\end{itemize}

这些想法没有认真评估转牌追牌价格。

\subsubsection*{理性分析}

第一，判断转牌类型。\texttt{4} 基本属于空白牌，没有完成你的同花，也没有明显改善你的具体牌力。

第二，判断你的牌力。你现在只有同花听牌，可能还有部分高张价值，但面对大盲突然大额下注，摊牌价值很低。

第三，计算继续条件。转牌只剩一张河牌，你完成同花的机会有限。如果对手下注接近底池，你需要较高胜率才能跟注。除非你认为对手在河牌同花完成时还会继续支付大量筹码，或者他的下注范围中有足够诈唬，否则跟注并不自动合理。

第四，考虑反向风险。如果你不是坚果同花听牌，即使河牌完成同花，也可能输给更高同花。同时，对手大额下注可能代表强 A、两对、暗三条或其他强牌，你的隐含赔率未必如想象中那么高。

\subsubsection*{推荐行动}

多数情况下倾向弃牌，除非你对对手有明确读牌，认为他有大量诈唬，或者你的听牌具有额外权益和足够隐含赔率。

\subsubsection*{本手牌教训}

转牌听牌不能只因为“还有机会”就继续。翻牌追牌和转牌追牌是完全不同的事情。到了转牌，只剩一张牌，面对大注时必须严格考虑价格。

\subsection{牌例四：空白转牌上的第二枪}

\subsubsection*{牌局背景}

有效筹码 100BB。你在按钮位拿到 \texttt{KQo}，前面玩家弃牌，你 open raise 到 2.5BB。大盲跟注。

翻牌为 \texttt{Q72 rainbow}。大盲过牌，你下注 1/3 底池，大盲跟注。

转牌为 \texttt{3}。大盲再次过牌，行动轮到你。

\subsubsection*{新手常见想法}

新手可能会想：

\begin{itemize}
\item “我有顶对，继续下注。”
\item “3 是空白牌，应该没什么变化。”
\item “他跟了翻牌，可能有 Q 或小对子。”
\end{itemize}

这些想法大致合理，但仍然需要明确下注目的。

\subsubsection*{理性分析}

第一，判断转牌类型。\texttt{3} 是空白转牌，基本没有改变 \texttt{Q72} 的牌面结构，也没有完成明显听牌。

第二，判断范围优势。按钮 open 对大盲防守，在 Q 高干燥面上通常拥有范围优势。转牌空白后，这个优势仍然存在。

第三，判断具体牌力。\texttt{KQ} 是强顶对，踢脚较好。它领先许多较弱 Q、7x、小对子和部分浮动牌。

第四，明确下注目的。继续下注主要是价值下注。你希望对手用 \texttt{QJ}、\texttt{QT}、\texttt{Q9}、7x、小对子等牌继续。由于牌面仍然干燥，你可以使用中等尺度，既获取价值，又不把较弱范围全部赶走。

第五，规划河牌。许多空白河牌可以继续考虑价值下注；如果河牌出现 A、K、7、同花完成牌或对手主动下注，需要重新评估。

\subsubsection*{推荐行动}

继续下注，使用小到中等尺度。

这次第二枪不是因为“空白牌就必须打”，而是因为你的牌力、范围优势和价值目标都支持继续。

\subsubsection*{本手牌教训}

空白转牌可以延续翻牌逻辑，但仍然要重新明确下注目的。合理的第二枪来自价值或诈唬依据，而不是机械习惯。

\section{本章总结}

转牌是整手牌中策略变化最明显的一条街。翻牌时很多牌力还处于模糊状态，但到了转牌，牌面结构、双方范围和听牌权益都会发生新的变化。

本章最重要的认知，是转牌必须重新评估。翻牌下注不代表转牌必须继续，翻牌跟注也不代表转牌必须继续跟。每一条街都是独立决策，都需要根据当前信息重新判断。

转牌出张大致可以分为三类：

\begin{center}
\textbf{强化我范围的牌：可以继续进攻。\\
强化对手范围的牌：需要谨慎。\\
空白牌：延续翻牌逻辑，但重新评估下注目的。}
\end{center}

当转牌强化你的范围时，你可以用强价值牌继续下注，也可以选择合适诈唬牌施压。当转牌强化对手范围时，即使你在翻牌拥有强牌，也需要降低进攻频率，更多考虑控池。当转牌是空白牌时，可以延续翻牌逻辑，但仍然要重新确认价值、诈唬和河牌计划。

防守方在转牌尤其要避免沉没成本思维。已经投入底池的筹码不再属于你，不能因为翻牌跟了，就在转牌无脑继续。转牌面对下注时，你需要重新评估牌力、对手范围、赔率和河牌计划。

听牌在转牌的价值明显不同于翻牌。翻牌还有两张牌可以改善，转牌只剩一张。面对大额下注时，普通听牌和弱听牌经常不能继续。

中等牌力在转牌上常常需要控池。它们有摊牌价值，但不适合承受多条街大额下注。学会让底池大小匹配牌力，是从娱乐玩家走向成熟牌手的重要一步。

下一章我们将进入河牌圈。河牌是最终决策街，没有未来权益，没有下一张牌可以等待。所有行动都将直接指向价值下注、诈唬下注、抓诈唬、摊牌或弃牌。

\section{课后训练}

\subsection*{训练一：转牌分类训练}

从最近的牌局或训练软件中记录 20 手进入转牌的牌局。对每一手牌的转牌进行分类：

\begin{itemize}
\item 强化我范围的转牌；
\item 强化对手范围的转牌；
\item 基本没有改变局势的空白转牌。
\end{itemize}

每手牌回答：

\begin{enumerate}
\item 这张转牌改变了什么？
\item 它更有利于谁的范围？
\item 我的具体牌力提升了、下降了，还是基本不变？
\item 我是否应该继续进攻、控池或放弃？
\end{enumerate}

\subsection*{训练二：第二枪检查}

记录 10 手你在转牌继续下注的牌，逐手回答：

\begin{itemize}
\item 这张转牌是否支持我继续下注？
\item 我下注是价值还是诈唬？
\item 如果是价值下注，哪些更差牌会跟？
\item 如果是诈唬下注，哪些更好牌会弃？
\item 如果被跟注，河牌我准备怎么办？
\end{itemize}

如果你发现很多第二枪只是因为“翻牌已经打了”，说明你存在机械开火的问题。

\subsection*{训练三：转牌弃牌训练}

记录 10 手你在翻牌跟注后，转牌面对下注的牌。分析其中哪些本该在转牌弃牌。

重点回答：

\begin{enumerate}
\item 转牌是否改善了我的牌？
\item 对手第二枪代表什么范围？
\item 我的牌是否只是在抓诈唬？
\item 当前下注尺度下，我是否有足够胜率？
\item 我是否因为已经投入筹码而不愿弃牌？
\end{enumerate}

训练目标是减少“翻牌跟了，转牌无脑继续”的习惯。

\subsection*{训练四：听牌价格训练}

选择 10 手你在转牌仍然持有听牌的手牌，记录：

\begin{itemize}
\item 听牌类型：同花听牌、两头顺、卡顺、组合听牌；
\item 是否为坚果听牌；
\item 面对下注尺度；
\item 直接赔率是否足够；
\item 是否有隐含赔率；
\item 是否有加注半诈唬空间。
\end{itemize}

不要只记录最后有没有中牌，而要判断转牌继续是否合理。

\subsection*{训练五：中等牌力控池训练}

找出最近 10 手你在转牌持有中等牌力的牌，例如弱顶对、第二对子、中等口袋对子。

逐手分析：

\begin{enumerate}
\item 我的牌适合打大底池吗？
\item 下注后更差牌会不会继续？
\item 更强牌会不会大量存在？
\item 如果被加注，我能否继续？
\item 过牌控池是否更合理？
\end{enumerate}

这个训练的目标，是帮助你学会让底池大小匹配牌力，而不是用中等牌力盲目打大底池。
