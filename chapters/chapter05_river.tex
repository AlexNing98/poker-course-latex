\chapter{河牌 / River}

\section{本章导入}

河牌是整手牌的最终决策街。翻前决定你带着什么范围进入底池，翻牌让双方第一次围绕公共牌展开较量，转牌重新塑造范围和牌力，而到了河牌，所有不确定的公共牌都已经发完。此时没有下一张牌可以等待，也没有未来权益可以实现。每一个行动都必须直接面对最终结果。

正因为河牌没有后续补牌机会，所以它的决策性质与翻牌、转牌明显不同。翻牌和转牌上的下注，有时可以包含保护、半诈唬、实现权益、为后续街铺垫等目的；但到了河牌，下注的逻辑会变得更加纯粹：如果你下注，要么是希望更差牌跟注，获得价值；要么是希望更好牌弃牌，完成诈唬。如果你过牌，要么是为了摊牌，要么是因为下注无法盈利。如果你面对下注，要么你的牌能击败对手足够多的下注范围而跟注，要么就应该弃牌。

很多新手在河牌最容易犯两个极端错误。第一是不舍得弃牌。前面已经投入了很多筹码，河牌面对大注时，他们会觉得“都打到这里了，不能弃”“万一他在诈唬呢”“我想看一眼”。第二是不敢薄价值下注。明明有机会让更差牌支付，却因为害怕被加注、害怕对手有更强牌，选择过牌摊牌，长期损失大量价值。

河牌也是错误成本最高的一条街。翻前错误可能只是损失几个盲注，翻牌错误可能损失一到两次下注，而河牌面对大注时，一次错误跟注往往会损失一个很大的底池。反过来，一次漏掉的价值下注，也可能让你长期少赢很多筹码。

本章的目标，是帮助你建立河牌最终决策框架。你需要理解河牌为什么没有未来权益，价值下注和诈唬下注如何判断，什么是摊牌价值，面对河牌下注如何决定是否跟注，阻断牌如何帮助判断范围，以及为什么河牌大注通常代表更极化的范围。

本章的核心输出是“河牌决策三问”：

\begin{center}
\textbf{我有摊牌价值吗？\\
我下注是价值还是诈唬？\\
面对下注，我的牌能击败对手足够多的范围吗？}
\end{center}

只要你能在河牌圈稳定回答这三个问题，就能避免许多娱乐玩家常见的高成本错误。

\section{核心概念}

\subsection{河牌没有未来权益，只有最终决策}

河牌最大的特点，是没有未来权益。所谓未来权益，是指你的牌在后续公共牌中继续改善的机会。翻牌上的同花听牌还有转牌和河牌两次机会，转牌上的顺子听牌还有河牌一次机会，但到了河牌，所有公共牌已经发完，牌力不会再变化。

因此，河牌上的每一个决策都必须直接面对当前牌力和对手范围。

如果你持有一手强牌，问题是：是否还有更差牌会跟注？如果答案是肯定的，你可以考虑价值下注。

如果你持有一手没有摊牌价值的弱牌，问题是：是否有更好的牌会因为你的下注而弃牌？如果答案是肯定的，你可以考虑诈唬下注。

如果你持有一手中等摊牌价值牌，问题是：下注后是否能被更差牌跟注？如果不能，那么过牌摊牌可能更合理。

如果你面对对手下注，问题是：你的牌能否击败对手下注范围中足够多的组合？如果不能，即使你很想看牌，也应该弃牌。

河牌不再适合用“再看一张”“还有机会”“万一中了”这样的语言思考。到了河牌，机会已经结束，剩下的只有最终决策。

\subsection{价值下注：更差的牌能否跟注？}

河牌价值下注的核心问题非常简单：

\begin{center}
\textbf{如果我下注，哪些更差的牌会跟注？}
\end{center}

如果你无法列出足够多会跟注的更差牌，那么这个下注就可能不是价值下注，而是在把自己暴露给更强牌。

例如，你持有 \hand{A♠ K♣}，公共牌为 \hand{A♠ 7♣ 2♦ K♥ 3♠}，对手一路跟注。河牌你拥有两对。如果对手范围中有大量较弱 A、Kx、两对以下的摊牌牌力，并且对手有跟注倾向，那么河牌价值下注通常合理。

但如果你持有 \hand{A♠ 9♣}，公共牌为 \hand{A♠ Q♣ J♦ 7♥ 8♠}，对手在翻牌和转牌都跟注，河牌牌面完成了很多顺子和两对组合。此时你虽然仍然有顶对，但如果下注后更差牌很少跟注，而更强牌大量存在，那么下注价值就很薄，甚至可能不成立。

价值下注不是“我可能领先，所以下注”。正确逻辑是：

\begin{enumerate}
\item 我认为自己领先对手相当一部分范围；
\item 对手会用足够多更差牌跟注；
\item 我的下注尺度能让这些更差牌支付；
\item 即使被跟注，我长期仍然能盈利。
\end{enumerate}

新手经常把强牌价值下注和中等牌力下注混在一起。强牌可以争取更多价值；中等牌力则需要谨慎判断是否有足够更差牌愿意支付。河牌价值下注越薄，就越需要了解对手类型。

面对跟注站型对手，可以更积极地进行薄价值下注，因为他们会用很多弱牌付钱。面对紧弱型对手，薄价值下注需要谨慎，因为他们河牌跟注范围更强。

\subsection{薄价值下注}

薄价值下注是河牌非常重要的进阶能力。所谓薄价值，是指你并不是持有绝对强牌，但你认为下注后仍然能被一些更差牌跟注，从而长期盈利。

例如，公共牌为 \hand{K♠ J♣ 7♦ 4♥ 2♠}，你持有 \hand{K♠ Q♣}。对手一路过牌跟注，河牌再次过牌。你的牌不是坚果，也可能输给 \hand{A♠ K♣}、\hand{K♠ J♣}、\hand{7♠ 7♣}、\hand{4♠ 4♣}、\hand{2♠ 2♣} 等强牌。但如果对手范围中有大量 \hand{K♠ T♣}、\hand{K♠ 9♣}、\hand{Jx} 或口袋对子愿意跟注小到中等下注，那么你就可以考虑薄价值下注。

薄价值下注的难点在于，它要求牌手既不盲目害怕，也不盲目自信。你需要清楚：

\begin{itemize}
\item 对手是否会用更差牌跟注；
\item 你的下注尺度是否让更差牌舒服跟注；
\item 对手是否会频繁 check-raise 河牌；
\item 你的牌在对手跟注范围中是否仍然领先足够多。
\end{itemize}

很多新手河牌价值不足，不是因为他们没有好牌，而是因为他们只敢用非常强的牌下注。长期来看，这会损失大量本可以获得的价值。优秀牌手会在合适对手面前，用第二强、第三强的合理牌力持续榨取薄价值。

\subsection{诈唬下注：更好的牌能否弃牌？}

河牌诈唬下注的核心问题同样简单：

\begin{center}
\textbf{如果我下注，哪些更好的牌会弃牌？}
\end{center}

如果对手不会弃掉任何比你好的牌，那么诈唬就没有意义。很多新手河牌诈唬失败，不是因为“运气不好被抓”，而是因为他们选择了错误的对象、错误的牌面、错误的尺度和错误的手牌。

一个合理的河牌诈唬通常需要满足几个条件：

\begin{itemize}
\item 你的牌几乎没有摊牌价值；
\item 你的前面行动能够代表强牌；
\item 这张河牌有利于你的范围，或完成了你可以代表的强牌；
\item 对手范围中有足够多中等牌力；
\item 对手有弃牌能力；
\item 你的下注尺度能让对手承受真实压力。
\end{itemize}

例如，你在翻前加注，翻牌和转牌持续下注，河牌完成前门同花。你的范围中确实包含许多强同花，而对手范围中有不少一对和两对类中等牌力。如果你持有一张高花阻断牌但没有摊牌价值，就可能具备一定诈唬价值。

相反，如果对手是典型跟注站，河牌拿着任何对子都不愿意弃，那么你的诈唬频率就应该大幅降低。对不会弃牌的人诈唬，是新手最昂贵的错误之一。

河牌诈唬不是为了“赢回底池”，也不是为了“证明自己敢打”。它必须建立在对手能弃牌、你的故事可信、你的手牌适合作为诈唬候选的基础上。

\subsection{摊牌价值与过牌摊牌}

摊牌价值指的是一手牌如果进入摊牌，有机会赢下底池的能力。并不是所有有摊牌价值的牌都应该下注。很多牌在河牌最好的选择，是过牌摊牌。

例如你持有第二对子，公共牌为 \hand{K♠ J♣ 7♦ 4♥ 2♠}。如果你下注，很少有更差牌跟注，而更强牌不会弃牌。那么下注既不是价值，也不是诈唬。此时过牌摊牌通常更好。

中等摊牌价值牌的特点是：

\begin{itemize}
\item 能击败对手的一些空气牌和错过听牌；
\item 可能输给对手的顶对、两对、暗三条等价值牌；
\item 下注后很难被更差牌支付；
\item 作为诈唬又无法让足够多更好牌弃牌。
\end{itemize}

这类牌在河牌最常见的目标，就是便宜摊牌。过牌不是放弃，而是让手牌进入它最适合的路径。

新手常犯的错误，是把所有能赢摊牌的牌都拿去下注。他们觉得“我可能领先，所以我要打”。但如果下注后只被更强牌跟注，更差牌全部弃牌，这个下注就是典型的价值切割自己。

判断是否应该过牌摊牌，可以问：

\begin{center}
\textbf{下注之后，我是让更差牌跟注，还是只让更强牌继续？}
\end{center}

如果答案偏向后者，过牌往往更好。

\subsection{面对河牌下注如何判断是否跟注}

面对河牌下注，尤其是大额下注，是新手最容易亏损的场景之一。原因在于，河牌已经没有未来权益，你跟注只能靠当前牌力赢下摊牌。如果对手价值牌比例太高，你的跟注就会长期亏损。

河牌跟注的核心问题是：

\begin{center}
\textbf{我的牌能击败对手下注范围中足够多的组合吗？}
\end{center}

注意，这里不是问“他有没有可能在诈唬”。任何对手都有可能诈唬。问题是，他的诈唬是否足够多，足以让你的跟注盈利。

面对河牌下注时，可以按以下步骤判断：

\begin{enumerate}
\item 先看下注尺度。下注越大，你需要赢的比例越高。
\item 列出对手可能的价值牌。包括强顶对、两对、暗三条、顺子、同花等。
\item 列出对手可能的诈唬牌。包括错过同花听牌、错过顺子听牌、没有摊牌价值的空气牌。
\item 判断对手类型。这个对手是否真的会用足够多错过牌诈唬？
\item 判断你的牌在防守范围中的位置。你是否持有足够强的 bluff-catcher？
\end{enumerate}

Bluff-catcher，即抓诈唬牌，指的是一手通常打不过对手的价值下注，但可以击败对手诈唬的牌。例如你持有中等对子，面对河牌大注，如果对手不是诈唬，你基本落后；如果对手在诈唬，你能赢。这类牌的跟注价值完全取决于对手诈唬频率是否足够。

新手面对河牌下注时，最危险的想法是：“我不信。”不信不是理由。你必须能说明对手有哪些诈唬组合，以及这些组合是否足够多。

\subsection{阻断牌的入门理解}

阻断牌，即 blocker，是指你手中的牌减少了对手持有某些组合的可能性。河牌决策中，阻断牌尤其重要，因为它可以帮助你判断对手价值牌和诈唬牌的组合数量，也可以帮助选择更合适的诈唬候选。

例如公共牌出现三张同花，你手中持有该花色的 A。即使你没有成同花，你也阻断了对手持有坚果同花的可能性。此时，如果你的行动线可以代表同花，你的手牌可能更适合作为诈唬候选。

再如，公共牌为 \hand{A♠ K♣ 7♦ 4♥ 2♠}，你持有一张 A。这意味着对手持有 \hand{A♠ A♣} 或强 A 的组合数量减少了一部分。阻断效果可以让你在某些河牌局面中更有信心判断对手范围。

但新手要注意，阻断牌不是万能理由。不能因为“我有一张 blocker”就随便诈唬或跟注。阻断牌只是辅助判断，它必须和牌面结构、行动线、对手类型、下注尺度结合使用。

入门阶段可以先记住三类常见阻断作用：

\begin{itemize}
\item 阻断坚果牌：例如持有同花 A，减少对手坚果同花组合；
\item 阻断强价值牌：例如持有 A 或 K，减少对手强顶对或暗三条组合；
\item 阻断对手弃牌范围时要谨慎：如果你持有的牌阻断了对手可能弃掉的弱牌，反而可能降低诈唬效果。
\end{itemize}

一个好的河牌诈唬候选，通常希望阻断对手的强价值牌，同时不阻断对手会弃掉的中等牌力。

\subsection{河牌大注为什么通常更极化}

河牌大注通常代表更极化的范围。所谓极化，是指下注范围主要由两端组成：一端是非常强的价值牌，另一端是没有摊牌价值、希望通过下注让对手弃牌的诈唬。中等牌力通常不适合大额下注。

原因很简单。河牌没有未来权益，大额下注会迫使对手用更强的范围继续。如果你用中等牌力大注，常常会让更差牌弃掉，让更强牌跟注或加注，结果把自己价值切割。真正适合大注的，是那些希望构建大底池的强牌，以及那些没有摊牌价值、只能靠弃牌权益获胜的诈唬牌。

例如公共牌为 \hand{A♠ K♣ 9♦ 5♥ 2♠}，你持有 \hand{A♠ K♣}，可以在合适情况下进行大额价值下注，因为你能击败许多对手会继续的强 A、两对或抓诈唬牌。相反，如果你持有 \hand{K♠ Q♣}，大额下注就很尴尬：更差牌未必跟注，更强牌却会继续。

面对河牌大注，防守方也要理解极化逻辑。你面对的不是“对手随便下注”，而是一个可能由强价值牌和诈唬组成的范围。你的任务是判断：

\begin{itemize}
\item 对手是否有足够强牌这样下注；
\item 对手是否也有足够诈唬这样下注；
\item 你的牌是否只是抓诈唬；
\item 当前价格下，你需要抓到诈唬的频率是多少；
\item 这个对手是否真的会用足够多诈唬平衡大注。
\end{itemize}

在低级别或娱乐局中，很多玩家河牌大注偏价值，诈唬不足。面对这类对手，新手应该更愿意弃掉边缘抓诈唬牌，而不是因为“他可能在 bluff”就轻易支付。

\section{新手常见误区}

\subsection{误区一：不舍得弃牌}

河牌不舍得弃，是娱乐玩家最昂贵的错误之一。很多人面对大注时，心里其实已经知道自己大概率落后，但仍然忍不住跟注。原因通常不是数学判断，而是情绪。

他们会想：

\begin{itemize}
\item “我都跟到河牌了，现在弃太亏。”
\item “万一他在诈唬呢？”
\item “我想看他到底有什么。”
\item “我不能总被人打走。”
\item “这个底池太大了，我不想放弃。”
\end{itemize}

这些想法都不能构成跟注理由。河牌跟注必须建立在对手诈唬足够多、你的牌能击败足够多下注范围的基础上。已经投入的筹码属于底池，不应该影响当前决策。

真正优秀的弃牌，常常发生在河牌。能在大底池中放下第二好牌，是成熟牌手的重要能力。

\subsection{误区二：只要有牌就想看摊牌}

有些新手面对河牌下注时，只要自己有一对，就觉得应该看一眼。尤其是顶对、两对或超对，很容易让他们产生“这么强不能弃”的心理。

但河牌牌力是相对的。顶对在某些牌面上很强，在另一些牌面上只是 bluff-catcher。两对在安全牌面上可能是价值牌，在四连顺或三同花牌面上可能只是中等牌力。

面对河牌下注时，不要问“我的牌看起来强不强”，而要问“我的牌能击败对手这样下注的哪些牌”。如果对手下注范围几乎全是更强价值牌，你有一对也该弃；如果对手有大量错过听牌，你的中等对子也可能可以跟注。

\subsection{误区三：河牌价值下注太少}

很多新手害怕河牌下注。他们会想：“万一他有更强牌怎么办？”“我过牌摊牌也能赢。”“我不想被加注。”结果是，他们漏掉了大量薄价值。

例如你持有强顶对，对手一路被动跟注，河牌出现空白牌。对手范围中有许多更差顶对、第二对子或抓诈唬牌。如果你过牌摊牌，虽然可能赢下底池，但你没有从这些更差牌中再多赢一注。

长期盈利不仅来自避免错误，也来自抓住价值。对喜欢跟注的对手，河牌薄价值下注是非常重要的盈利来源。

\subsection{误区四：诈唬对象选错}

河牌诈唬最怕选错对象。有些玩家根本不喜欢弃牌，不管牌面多危险，只要有一对就想看摊牌。对这类玩家，复杂诈唬通常没有必要。你应该减少诈唬，增加价值下注。

另一些玩家过于谨慎，面对大额下注容易弃掉中等牌力。对这类玩家，合理诈唬可以增加盈利。

新手经常反过来：对跟注站拼命诈唬，对紧弱玩家只敢价值下注。这会让策略完全失衡。

河牌诈唬前，先问：

\begin{center}
\textbf{这个对手真的会弃牌吗？}
\end{center}

如果答案是否定的，再漂亮的诈唬故事也没有意义。

\subsection{误区五：用有摊牌价值的牌转诈唬}

很多中等牌力在河牌有摊牌价值，但不适合下注。新手有时会把这类牌拿去诈唬，结果反而浪费了摊牌价值。

例如你持有第二对子，对手错过听牌时你可能已经能赢。如果你下注，可能让对手的错过听牌弃牌，而这些牌本来就会输给你；同时，对手更强牌会继续。这样下注既没有价值，也没有必要诈唬。

适合诈唬的牌，通常是几乎没有摊牌价值的牌。因为如果不下注，你很难赢；下注至少有机会让更好牌弃掉。有摊牌价值的牌，很多时候应该选择过牌摊牌。

\subsection{误区六：忽视下注尺度}

河牌下注尺度会直接影响对手继续范围。小注可能被很多较弱牌跟注，适合薄价值；大注会让对手范围收紧，更适合极化策略。

新手常常不区分尺度。强牌随便打一个习惯尺度，诈唬也随便打一个习惯尺度，结果下注目的和尺度不匹配。

如果你是薄价值下注，通常希望使用让更差牌愿意跟注的尺度。
如果你是极化价值下注或诈唬下注，则需要使用能制造足够压力的尺度。
如果你的牌只是中等摊牌价值，可能根本不该下注。

尺度不是装饰，而是策略的一部分。

\section{实战思考框架}

本章的核心框架是“河牌决策三问”：

\begin{center}
\textbf{我有摊牌价值吗？\\
我下注是价值还是诈唬？\\
面对下注，我的牌能击败对手足够多的范围吗？}
\end{center}

\subsection{第一问：我有摊牌价值吗？}

河牌第一步，是判断你的牌是否有摊牌价值。

如果你的牌能击败对手一定比例的过牌范围或错过听牌，那么它就有摊牌价值。比如第二对子、小口袋对子、弱顶对，甚至有时 A 高，都可能在特定局面具有摊牌价值。

有摊牌价值的牌，不一定要下注。很多时候，它们最适合过牌摊牌。因为下注后，更差牌会弃牌，更强牌会继续，你的下注无法盈利。

如果你的牌几乎没有摊牌价值，例如错过的同花听牌、错过的顺子听牌、完全空气牌，那么你只有两个选择：放弃，或者在合适条件下诈唬。是否诈唬，要看你能否让足够多更好牌弃牌。

判断摊牌价值时，可以问：

\begin{itemize}
\item 如果双方过牌摊牌，我大概能赢哪些牌？
\item 对手范围中是否有足够多错过听牌或弱牌？
\item 我的牌是否太弱，几乎无法摊牌获胜？
\item 如果我下注，会不会浪费本来能赢的摊牌价值？
\end{itemize}

\subsection{第二问：我下注是价值还是诈唬？}

如果你考虑下注，必须明确它属于价值还是诈唬。

价值下注的判断标准是：更差牌能否跟注。
诈唬下注的判断标准是：更好牌能否弃牌。

如果一手牌下注后，更差牌很少跟注，更好牌也很少弃牌，那么这个下注通常没有意义。

河牌下注前，可以使用以下检查表：

\begin{enumerate}
\item 我希望对手跟注还是弃牌？
\item 如果希望对手跟注，他有哪些更差牌？
\item 如果希望对手弃牌，他有哪些更好牌？
\item 我的下注尺度是否符合这个目的？
\item 我的前面行动线是否支持我代表这类牌？
\end{enumerate}

如果这些问题回答不清楚，过牌通常比下注更好。

\subsection{第三问：面对下注，我的牌能击败对手足够多的范围吗？}

当你面对河牌下注时，要把注意力从自己的牌转移到对手下注范围。

你需要列出两类牌：

\begin{itemize}
\item 对手的价值牌；
\item 对手的诈唬牌。
\end{itemize}

然后判断你的牌能击败哪些。很多河牌跟注牌并不能击败对手的价值牌，只能击败诈唬。这种牌就是 bluff-catcher。拿着 bluff-catcher 跟注是否正确，取决于对手诈唬是否足够多。

面对下注时，建议按以下顺序思考：

\begin{enumerate}
\item 这个下注尺度代表什么压力？
\item 对手有哪些合理价值牌？
\item 对手有哪些合理诈唬牌？
\item 对手是否真的会用这些牌诈唬？
\item 我的牌在所有可跟注牌中处于什么位置？
\end{enumerate}

如果对手价值牌很多、诈唬很少，而你的牌只能抓诈唬，那么弃牌通常是正确选择。

\subsection{河牌行动选择表}

为了帮助新手快速判断，可以使用以下简化表格：

\begin{center}
\begin{tabular}{lll}
\toprule
\textbf{手牌类型} & \textbf{主要问题} & \textbf{常见行动} \\
\midrule
强价值牌 & 更差牌能否支付？ & 下注、加注 \\
薄价值牌 & 对手是否会用更差牌跟？ & 小到中等下注 \\
中等摊牌价值 & 下注是否会被更差牌跟？ & 过牌摊牌、跟注小注 \\
弱摊牌价值 & 能否击败诈唬？ & 过牌、谨慎跟注或弃牌 \\
无摊牌价值空气牌 & 更好牌会不会弃？ & 诈唬或放弃 \\
\bottomrule
\end{tabular}
\end{center}

这张表不是固定答案，而是提醒你：不同牌力在河牌承担不同功能。不要用中等牌力做强价值下注，也不要用有摊牌价值的牌随意转诈唬。

\section{牌例拆解}

\subsection{牌例一：强顶对的河牌价值下注}

\subsubsection*{牌局背景}

有效筹码 100BB。你在按钮位拿到 \hand{AKo}，前面玩家弃牌，你 open raise 到 2.5BB。大盲跟注。

翻牌为 \hand{A♠ 7♣ 2♦ rainbow}。大盲过牌，你下注 1/3 底池，大盲跟注。

转牌为 \hand{K♠}。大盲过牌，你下注 2/3 底池，大盲跟注。

河牌为 \hand{3♠}。大盲再次过牌，行动轮到你。

\subsubsection*{新手常见想法}

很多新手会想：

\begin{itemize}
\item “我已经两对了，应该下注。”
\item “他跟了两条街，会不会有暗三条？”
\item “如果我过牌也能赢，要不要保守一点？”
\end{itemize}

这手牌需要判断的是：河牌下注后，是否有足够多更差牌会跟注。

\subsubsection*{理性分析}

第一，判断摊牌价值。你持有 \hand{A♠ K♣}，在 \hand{A♠ 7♣ 2♦ K♥ 3♠} 上是很强的两对，摊牌价值很高。

第二，判断是否价值下注。对手大盲跟注翻牌和转牌后，范围中可能有较弱 A，例如 \hand{A♠ Q♣}、\hand{A♠ J♣}、\hand{A♠ T♣}、\hand{A♠ 9♣}，也可能有一些 Kx、7x 或不愿弃牌的口袋对子。河牌 3 基本是空白牌，没有完成明显听牌。许多更差 A 可能会继续支付。

第三，判断风险。你确实会输给 \hand{7♠ 7♣}、\hand{2♠ 2♣}、\hand{3♠ 3♣}、\hand{A♠ 7♣}、\hand{A♠ 2♣}、\hand{A♠ 3♣} 等牌，但这些组合数量有限，而且对手并不一定全部以被动跟注方式打到河牌。

第四，选择尺度。因为你希望更差 A 跟注，可以选择中等偏大的价值下注。若对手是跟注站，可以下注更大；若对手偏紧，可以使用更容易被跟注的中等尺度。

\subsubsection*{推荐行动}

下注，进行价值下注。

这不是因为“我有两对所以必须打”，而是因为对手范围中存在足够多更差牌可以跟注。

\subsubsection*{本手牌教训}

河牌价值下注的核心不是自己牌看起来多强，而是更差牌能否支付。在空白河牌上，强两对通常可以向较弱 A 和其他摊牌牌力继续获取价值。

\subsection{牌例二：有摊牌价值，不适合转诈唬}

\subsubsection*{牌局背景}

有效筹码 100BB。你在 cutoff 拿到 \hand{Q♠ J♠}，open raise 到 2.5BB。大盲跟注。

翻牌为 \hand{K♠ J♠ 7♦ two-tone}。大盲过牌，你下注 1/3 底池，大盲跟注。

转牌为 \hand{4♠}。双方过牌。

河牌为 \hand{2♠}，没有完成同花。大盲过牌，行动轮到你。

\subsubsection*{新手常见想法}

一些新手会想：

\begin{itemize}
\item “我只有 J，一下注也许能把 K 打走。”
\item “他转牌过牌，可能不强。”
\item “我不想摊牌输给小 K。”
\end{itemize}

问题在于，你的 \hand{J} 对子本身有一定摊牌价值，不一定适合拿来诈唬。

\subsubsection*{理性分析}

第一，判断摊牌价值。你有第二对子，可以击败对手错过的同花听牌、顺子听牌、部分 7x 和一些浮动牌。因此，你不是完全空气牌。

第二，判断价值下注。下注后，哪些更差牌会跟注？可能有一些 Jx 或 7x，但数量不一定多。很多错过听牌会直接弃牌，而这些牌本来就输给你。

第三，判断诈唬下注。你希望打掉哪些更好牌？主要是弱 K。但大盲翻牌跟注、转牌过牌后，很多 Kx 仍然可能不愿意弃，尤其面对小到中等下注。若你打大注试图让 Kx 弃牌，又要冒较大风险。

第四，比较过牌。过牌可以直接摊牌，保留你对错过听牌和弱牌的胜率。下注反而可能让更差牌弃掉、更强牌继续。

\subsubsection*{推荐行动}

过牌摊牌。

这手牌有一定摊牌价值，不适合轻易转成诈唬。除非你非常确定对手会弃掉大量 Kx，否则下注意义不大。

\subsubsection*{本手牌教训}

有摊牌价值的牌，不要随意转诈唬。河牌过牌摊牌是一种合理行动，尤其当下注无法让更差牌跟注，也无法让足够多更好牌弃牌时。

\subsection{牌例三：面对河牌大注的抓诈唬}

\subsubsection*{牌局背景}

有效筹码 100BB。你在按钮位拿到 \hand{K♠ Q♣}，open raise 到 2.5BB。大盲跟注。

翻牌为 \hand{Q♠ 8♣ 2♦ rainbow}。大盲过牌，你下注 1/3 底池，大盲跟注。

转牌为 \hand{5♠}，带来后门同花听牌。大盲过牌，你下注 2/3 底池，大盲跟注。

河牌为 \hand{J♠}，没有完成同花。大盲突然下注接近底池。

\subsubsection*{新手常见想法}

很多新手会想：

\begin{itemize}
\item “我有顶对，不能随便弃。”
\item “他可能错过听牌在 bluff。”
\item “我想看他到底是什么。”
\item “如果每次都弃，会不会太弱？”
\end{itemize}

这些想法都不足以直接支持跟注。你需要分析对手河牌下注范围。

\subsubsection*{理性分析}

第一，判断你的牌。你持有 \hand{K♠ Q♣}，是顶对好踢脚。它能击败部分诈唬和较弱 Q，但面对河牌接近底池的主动下注，已经接近 bluff-catcher。

第二，列出对手价值牌。大盲可能有 \hand{Q♠ J♣} 河牌两对，\hand{J♠ 8♣}、\hand{J♠ 5♣} 这类少量两对，\hand{8♠ 8♣}、\hand{2♠ 2♣}、\hand{5♠ 5♣} 暗三条，也可能有慢打的强 Q 或其他特殊组合。

第三，列出对手诈唬牌。可能包括错过的后门同花听牌、\hand{T♠ 9♣}、\hand{9♠ 7♣}、部分没有摊牌价值的浮动牌。但需要判断对手是否真的会在河牌用这些牌接近底池下注。

第四，判断对手类型。如果对手是被动娱乐玩家，河牌突然大注往往偏价值，诈唬不足。此时即使你有顶对，也应该谨慎。如果对手是激进玩家，能用错过听牌和阻断牌施压，那么跟注频率可以提高。

\subsubsection*{推荐行动}

取决于对手类型。面对被动玩家，倾向弃牌；面对能充分诈唬的激进玩家，可以考虑跟注。

\subsubsection*{本手牌教训}

河牌跟注不能只因为“我有顶对”或“他可能诈唬”。你必须判断对手的价值牌和诈唬牌比例。很多时候，顶对在河牌大注面前只是抓诈唬牌。

\subsection{牌例四：错过听牌的河牌诈唬}

\subsubsection*{牌局背景}

有效筹码 100BB。你在按钮位拿到 \hand{A♠ 5♠}，前面玩家弃牌，你 open raise 到 2.5BB。大盲跟注。

翻牌为 \hand{K♠ 7♣ 2♦}，其中两张与你同花。大盲过牌，你下注 1/3 底池，大盲跟注。

转牌为 \hand{Q♠}，没有完成同花。大盲过牌，你继续下注 2/3 底池，大盲跟注。

河牌为 \hand{3♠}，同花没有完成。大盲过牌，行动轮到你。

\subsubsection*{新手常见想法}

新手可能会想：

\begin{itemize}
\item “我没中，只能放弃。”
\item “我已经打了两枪，不打第三枪太可惜。”
\item “他都跟了两枪，应该不会弃。”
\end{itemize}

这手牌需要根据河牌、对手范围和阻断牌判断是否适合诈唬。

\subsubsection*{理性分析}

第一，判断摊牌价值。你持有 \hand{A♠ 5♠}，同花没有完成，几乎没有摊牌价值。如果过牌，大概率输给对手的 Kx、Qx、7x 或口袋对子。

第二，判断可代表强牌。你翻牌和转牌连续下注，在 \hand{K♠ 7♣ 2♦ Q♥ 3♠} 上可以代表 \hand{K♠ Q♣}、\hand{A♠ A♣}、\hand{A♠ K♣}、\hand{K♠ J♣}、暗三条等强牌。如果你的前面行动线合理，河牌继续下注有故事基础。

第三，判断对手范围。大盲跟注两枪后，范围中可能有 Kx、Qx、部分口袋对子和错过听牌。如果对手有弃牌能力，一些中等 K、Q 或口袋对子可能面对大注承受压力。

第四，判断阻断牌。你持有同花 A，阻断了对手部分坚果同花听牌，但由于同花未完成，这个阻断作用有限。若你的 A 还阻断部分 \hand{A♠ K♣}、\hand{A♠ A♣} 组合，也可能略微减少对手强价值牌数量。但你也要注意，你持有 A 可能阻断对手一些会弃牌的 A 高浮动牌。

第五，判断对手类型。如果大盲是跟注站，河牌诈唬不理想；如果大盲能弃掉中等牌力，且你的下注尺度足够制造压力，可以考虑诈唬。

\subsubsection*{推荐行动}

根据对手类型选择。面对有弃牌能力的对手，可以使用较大尺度诈唬；面对跟注站，过牌放弃更好。

\subsubsection*{本手牌教训}

河牌诈唬适合没有摊牌价值的牌，但必须满足对手能弃牌、行动线可信、牌面支持和尺度合理。不能因为“已经打了两枪”就机械开第三枪，也不能因为错过听牌就永远放弃。

\subsection{牌例五：漏掉薄价值的典型场景}

\subsubsection*{牌局背景}

有效筹码 100BB。你在按钮位拿到 \hand{K♠ Q♣}，open raise 到 2.5BB。大盲跟注。

翻牌为 \hand{K♠ 8♣ 3♦ rainbow}。大盲过牌，你下注 1/3 底池，大盲跟注。

转牌为 \hand{6♠}。双方过牌。

河牌为 \hand{2♠}。大盲过牌，行动轮到你。

\subsubsection*{新手常见想法}

很多新手会想：

\begin{itemize}
\item “我有顶对，过牌应该能赢。”
\item “如果他有 \hand{K♠ 8♣} 或暗三条，我下注会输钱。”
\item “已经到河牌了，没必要冒险。”
\end{itemize}

这些想法忽视了更差牌支付的可能性。

\subsubsection*{理性分析}

第一，判断摊牌价值。\hand{K♠ Q♣} 在 \hand{K♠ 8♣ 3♦ 6♥ 2♠} 上是很好的顶对。牌面较干燥，河牌没有完成明显听牌。

第二，判断价值下注。对手大盲可能用较弱 K、8x、口袋对子、甚至部分 A 高跟注翻牌。转牌双方过牌后，对手范围并不一定很强。河牌面对一个小到中等下注，较弱 K 和一些好奇跟注牌可能会支付。

第三，判断风险。你会输给 \hand{K♠ 8♣}、\hand{K♠ 6♣}、\hand{K♠ 2♣}、暗三条等，但这些组合并不占据对手大部分范围。更重要的是，如果你总是过牌摊牌，会漏掉大量来自较弱 K 和中等牌力的价值。

第四，选择尺度。由于这是薄价值下注，尺度不宜过大。小到中等下注更容易被更差牌跟注。

\subsubsection*{推荐行动}

下注，进行薄价值下注。

下注尺度可以偏小到中等，目标是让较弱 K、8x 或部分口袋对子跟注。

\subsubsection*{本手牌教训}

河牌不下注也能赢，并不代表过牌最好。如果更差牌会支付，你应该争取价值。薄价值下注是长期盈利的重要来源。

\section{本章总结}

河牌是最终决策街。到了河牌，没有后续补牌机会，没有未来权益，所有行动都直接指向摊牌价值、价值下注、诈唬下注或弃牌。

本章最重要的框架是“河牌决策三问”：

\begin{center}
\textbf{我有摊牌价值吗？\\
我下注是价值还是诈唬？\\
面对下注，我的牌能击败对手足够多的范围吗？}
\end{center}

河牌价值下注的核心，是更差牌能否跟注。不要因为自己“可能领先”就盲目下注，也不要因为害怕少数更强牌就错过薄价值。面对喜欢跟注的对手，合理扩大薄价值下注，是非常重要的盈利来源。

河牌诈唬下注的核心，是更好牌能否弃牌。诈唬不是情绪表达，也不是追回底池的手段。它必须建立在对手有弃牌能力、你的行动线可信、牌面支持、手牌适合作为诈唬候选的基础上。

有摊牌价值的牌，很多时候应该过牌摊牌。中等牌力如果下注后只让更差牌弃掉、更强牌继续，就没有下注意义。不要把本来可以摊牌获胜的牌，随意转化成低质量诈唬。

面对河牌下注，尤其是大注，不能只问“他有没有可能诈唬”。正确问题是：他的诈唬是否足够多？我的牌能否击败他下注范围中足够多的组合？很多河牌跟注牌只是 bluff-catcher，是否跟注取决于对手价值牌和诈唬牌比例。

河牌错误成本很高。一次错误跟注，往往会损失一个大底池；一次漏掉的价值下注，也会在长期中积累巨大损失。学会在河牌清晰地区分价值、诈唬、摊牌和弃牌，是从初学者走向成熟牌手的关键一步。

至此，我们已经拆解了一手牌从翻前到河牌的完整处理流程。下一章开始，我们将从单手牌流程上升到更整体的牌力处理框架，讨论不同牌力在范围中承担的功能，以及如何根据牌力类型制定更加稳定的实战策略。

\section{课后训练}

\subsection*{训练一：河牌三问记录}

下一次打牌后，记录至少 10 手进入河牌的手牌。每一手都回答：

\begin{enumerate}
\item 我有摊牌价值吗？
\item 如果我下注，是价值还是诈唬？
\item 如果我面对下注，我的牌能击败对手足够多的下注范围吗？
\end{enumerate}

训练目标是让你在河牌不再凭感觉行动，而是回到明确问题。

\subsection*{训练二：价值下注检查}

找出最近 10 手你在河牌下注的牌，逐手分析：

\begin{itemize}
\item 我下注的目标是什么？
\item 哪些更差牌会跟注？
\item 我的下注尺度是否让这些更差牌愿意跟注？
\item 如果被跟注，我是否大多数时候领先？
\item 这手牌是强价值还是薄价值？
\end{itemize}

如果你无法列出更差跟注牌，说明这次下注可能不是好的价值下注。

\subsection*{训练三：漏掉薄价值复盘}

找出最近 10 手你河牌选择过牌摊牌并赢下底池的手牌，分析其中哪些本可以下注。

逐手回答：

\begin{enumerate}
\item 对手是否可能用更差牌跟注？
\item 我的牌是否足够领先对手过牌范围？
\item 如果下注，应该使用什么尺度？
\item 我当时为什么没有下注？
\item 是合理控池，还是害怕导致漏价值？
\end{enumerate}

这个训练可以帮助你减少河牌价值不足的问题。

\subsection*{训练四：河牌跟注复盘}

记录最近 10 手你在河牌面对下注选择跟注的牌，分析：

\begin{itemize}
\item 对手下注尺度是多少？
\item 对手有哪些合理价值牌？
\item 对手有哪些合理诈唬牌？
\item 我的牌能击败哪些下注组合？
\item 我是否只是因为“不想弃”而跟注？
\end{itemize}

如果你发现很多跟注只基于“他可能诈唬”，说明你需要加强范围分析和弃牌纪律。

\subsection*{训练五：诈唬对象选择}

找出最近 5 手你河牌诈唬的牌，逐手分析：

\begin{enumerate}
\item 我的牌是否几乎没有摊牌价值？
\item 我的行动线是否能代表强牌？
\item 河牌是否有利于我的范围？
\item 对手是否有弃牌能力？
\item 我的下注尺度是否能让更好牌弃牌？
\end{enumerate}

如果对手是明显跟注站，且你没有足够可信的强牌故事，那么这类诈唬应当减少。

\subsection*{训练六：阻断牌入门练习}

选择 10 个河牌局面，记录你手中的两张牌，并分析它们是否阻断了对手的某些强牌或诈唬牌。

重点回答：

\begin{itemize}
\item 我是否阻断对手的坚果牌？
\item 我是否阻断对手的强价值牌？
\item 我是否阻断了对手可能弃牌的弱牌？
\item 这些阻断效果是否支持我诈唬、跟注或弃牌？
\end{itemize}

训练目标不是滥用 blocker，而是学会用它辅助判断组合数量。
