\chapter{如何成为优秀的牌手}

\section{本章导入}

很多初学者刚接触德州扑克时，最关心的问题通常是：“什么牌可以玩？”“翻牌中了顶对该不该打？”“河牌被下注还能不能跟？”这些问题当然重要，但它们还不是扑克学习的起点。真正的起点，是先理解德州扑克到底是一项什么样的游戏。

表面上看，德州扑克是一项纸牌游戏；更进一步看，它是一项关于概率、信息、心理和资金管理的综合竞技。每一手牌都发生在不完整信息之下，牌手无法知道对手的底牌，只能根据位置、行动、下注尺度、公共牌结构和对手倾向，推断对手可能持有的范围，并在有限信息中做出长期期望值最高的决策。

初学者最常见的问题，不是完全不懂技术，而是没有稳定的决策系统。拿到好牌时兴奋，拿到边缘牌时好奇，被下注时不甘心，连续输牌后急于追回损失，赢了几手后又开始放松纪律。于是，牌桌上的每一个决定都被情绪牵着走。这样的打法即使偶尔赢钱，也很难长期稳定盈利。

本章不是具体技术章，而是建立学习扑克的底层认知。我们先不急着讨论某一手牌到底该下注还是该弃牌，而是先回答几个更基础的问题：为什么扑克是一项竞技运动？为什么体能、心态和技术缺一不可？为什么很多人懂一些理论却依然打不好？为什么不能只靠感觉打牌？新手应该如何训练，才能真正进入水平提升的快车道？

如果把整门课程比作一栋建筑，那么本章就是地基。后续所有翻前、翻牌、转牌和河牌的技术内容，都需要建立在稳定状态、正确框架、持续复盘和长期执行之上。

本章的核心公式是：

\begin{center}
\textbf{优秀牌手成长公式：稳定状态 $\times$ 正确框架 $\times$ 持续复盘 $\times$ 长期执行}
\end{center}

这四个要素缺一不可。没有稳定状态，技术无法发挥；没有正确框架，练习只是在重复错误；没有持续复盘，经验无法转化为能力；没有长期执行，任何短期波动都会摧毁信心。

\section{核心概念}

\subsection{扑克为什么是一项竞技运动}

很多人误以为德州扑克只是运气游戏，因为短期结果确实会受到发牌影响。你可能用一手优势牌被对手河牌反超，也可能用一手边缘牌意外赢下大底池。短期来看，结果经常不完全受个人控制。

但竞技的本质，并不是每一次结果都完全由能力决定，而是在大量重复决策中，能力能够持续影响长期结果。德州扑克正是如此。单手牌可能有运气成分，但长期成千上万手牌之后，真正决定结果的是牌手的决策质量。

优秀牌手和娱乐玩家的区别，不在于优秀牌手每一手都能赢，而在于他们能在相似局面中反复做出更高期望值的选择。他们知道什么时候该进攻，什么时候该放弃，什么时候该薄价值下注，什么时候该控制底池，什么时候该抓诈唬，什么时候即使不甘心也必须弃牌。

因此，德州扑克不是“猜对手有没有牌”的游戏，而是“在不完整信息下持续做出高质量决策”的竞技项目。它考验的不只是牌技，还包括专注力、情绪稳定性、身体状态、学习能力、复盘能力和风险控制能力。

\subsection{优秀牌手的三大基础：体能、心态、技术}

很多初学者认为，只要学会足够多的技术，就能自然打好牌。但在真实牌桌上，技术并不会自动转化为行动。一个人可能知道翻前不能入池太宽，却在无聊时忍不住玩边缘牌；可能知道河牌面对大注应该理性分析，却在被对手连续施压后情绪化跟注；可能知道资金管理重要，却在输钱后升级别追损。

所以，优秀牌手至少需要三项基础能力：体能、心态和技术。

第一是体能。扑克看似坐着不动，但长时间保持专注本身就是消耗。疲劳会让人降低判断质量，忽略细节，放松纪律，过度简化决策。很多错误并不是因为牌手完全不知道该怎么打，而是因为在疲劳状态下，他没有精力继续执行正确策略。

第二是心态。扑克中最危险的状态之一是情绪接管决策。输一手大底池后想马上打回来，被对手诈唬后想证明自己，连续弃牌后想找一手牌“玩一下”，这些都是情绪驱动。优秀牌手并不是没有情绪，而是能识别情绪，并且不让情绪决定行动。

第三是技术。技术包括翻前范围、位置意识、牌面结构判断、下注目的、底池赔率、范围分析、对手倾向识别等内容。技术是扑克水平的核心，但技术必须建立在良好的状态和执行力之上。否则，懂再多概念，也可能在牌桌上变成凭感觉乱打。

可以说，体能决定你能否保持专注，心态决定你能否稳定执行，技术决定你知道如何决策。三者共同构成一名牌手的基本盘。

\subsection{情绪失控如何摧毁技术决策}

在扑克中，情绪失控通常被称为上头或 tilt。它不一定表现为明显的愤怒，也可能表现为急躁、不耐烦、侥幸、报复心理或过度自信。

例如，某位新手翻前拿到 \texttt{AA}，对手用一手同花连张跟注，并在河牌完成顺子赢下大底池。新手可能会觉得“他凭什么这样打还赢？”接下来几手牌，他开始放宽入池范围，频繁加注，试图把输掉的钱追回来。此时，他已经不是在根据牌局做决策，而是在根据情绪做决策。

情绪失控会带来几个典型后果。

第一，入池范围变宽。牌手会开始玩一些平时知道不该玩的手牌，理由是“不能一直弃牌”或者“这手看起来有机会”。

第二，跟注频率上升。面对下注时，牌手不再冷静分析对手范围和底池赔率，而是想着“他不可能每次都有”“我不想再被诈唬”。

第三，诈唬变得随意。情绪化牌手可能会在不合适的牌面、不合适的对手、不合适的尺度下强行诈唬，只是为了发泄压力。

第四，资金管理失效。输钱后升级别、加买入、延长牌局时间，都是情绪影响风险控制的表现。

真正危险的是，情绪失控时，牌手往往并不觉得自己正在失控。他会为自己的行为寻找理由，认为自己是在调整打法、惩罚对手、争取主动。实际上，他已经偏离了理性决策。

所以，学习扑克的第一步不是消灭情绪，而是建立一个原则：情绪可以出现，但不能指挥行动。

\subsection{长期主义与 EV 思维}

德州扑克是一项长期游戏。单手牌的输赢不能证明打法正确与否，一晚的结果也不能完全代表水平高低。判断一个决策是否优秀，不能只看这次结果，而要看它在长期重复中是否具有正期望值。

这里需要引入一个重要概念：EV，即 Expected Value，中文常译为期望值。简单来说，EV 指的是某个决策在长期重复中平均能带来的收益。

例如，假设某个跟注在长期看平均能盈利，那么即使这一次跟注后输掉了底池，这个跟注仍然可能是正确的。反过来，如果某个糟糕的跟注偶然赢了，也不能说明它是好决策。扑克学习中最重要的转变之一，就是从“结果导向”转向“决策导向”。

新手最容易犯的错误，是用结果评价过程：

\begin{itemize}
\item 赢了，就觉得自己打得好；
\item 输了，就觉得自己打错了；
\item 诈唬成功，就觉得诈唬合理；
\item 被抓诈唬，就觉得不该诈唬；
\item 跟注赢了，就觉得自己读牌准；
\item 跟注输了，就觉得自己运气差。
\end{itemize}

这种评价方式会严重阻碍进步。正确的复盘方式应该是：在当时的信息条件下，我的决策是否合理？我的范围判断是否准确？我的下注目的是否清晰？我的尺度是否合适？我的跟注是否满足数学条件？我的弃牌是否过紧或过松？

长期主义并不是让你忽视结果，而是让你用更长的样本看待结果。短期波动无法避免，但长期决策质量可以控制。优秀牌手追求的不是每一手牌都赢，而是在大量牌局中持续做出正 EV 决策。

\subsection{训练不是单纯多打牌}

很多新手以为，打得越多就会自然变强。事实上，如果没有正确框架和复盘习惯，单纯多打牌很可能只是不断重复错误。

错误经验积累得越多，反而越难改。比如，一个玩家长期习惯翻前用过宽范围跟注，偶尔用边缘牌中到大牌赢了底池，他就会强化这种坏习惯。下一次遇到类似局面，他会记住那次赢钱的经历，而不是记住长期亏损的事实。

有效训练至少包括三个环节。

第一，带着目标打牌。每一次练习不应该只是“随便打几手”，而应该有明确训练主题。例如，本周只重点训练翻前纪律；下一阶段重点训练翻牌 C-bet；再下一阶段重点训练河牌弃牌能力。

第二，记录关键手牌。不是每一手都需要复盘，但所有大底池、困难决策、情绪波动明显的手牌，都值得记录下来。复盘时不要只写“我输了”或“我被河杀”，而要记录位置、有效筹码、翻前行动、公共牌、下注尺度和自己的思考过程。

第三，复盘决策过程。复盘的重点不是重新感受输赢，而是重新分析决策质量。尤其要问：如果不知道对手底牌，只根据当时信息，我的行动是否合理？如果重来一次，我会不会做出同样选择？

真正的训练不是“打更多牌”，而是“用正确方法重复高质量决策”。

\subsection{资金管理与风险控制}

扑克中再优秀的策略，也无法消除短期波动。如果没有资金管理，即使牌手技术不错，也可能因为几次正常波动而被迫停止游戏，甚至情绪崩溃。

资金管理的核心目的不是让你永远不输钱，而是保证你在正常波动下仍然能够继续执行策略。它是一种风险控制系统。

对于初学者来说，最重要的原则有三条。

第一，只使用可以承受损失的资金。扑克训练资金必须和生活资金分开，不能用房租、学费、生活费或其他必要资金参与牌局。否则，每一次输赢都会带来过高心理压力，直接影响决策质量。

第二，选择适合自己资金量和水平的级别。很多新手升级别不是因为技术已经准备好，而是因为想赢得更快，或者想把之前输的钱追回来。这是非常危险的。级别越高，对手越强，波动越大，错误成本也越高。

第三，设置止损和停止条件。停止条件不应该只根据输赢决定，也应该根据状态决定。当你明显疲劳、急躁、不耐烦、想追损、开始随意入池时，即使还没输很多，也应该停止。因为此时继续打牌的最大风险不是运气不好，而是你已经不再能执行正确策略。

资金管理不是保守，而是职业化。它让牌手有能力穿越波动，而不是被短期结果摧毁。

\section{新手常见误区}

\subsection{误区一：觉得懂理论就等于会打牌}

很多新手看过一些文章、视频或起手牌表后，会觉得自己已经理解了扑克。但真正上桌后，一遇到复杂局面就会回到原来的直觉打法。

这是因为理论知识和实战能力之间存在距离。知道“位置很重要”，不等于你真的能在前位弃掉边缘牌；知道“不要过度跟注”，不等于你河牌面对大注时真的能弃掉顶对；知道“下注要有目的”，不等于你每次下注前都会问自己是价值还是诈唬。

理论只有经过反复训练、复盘和执行，才能转化成稳定能力。新手不要满足于“我听懂了”，而要追问：“我能不能在压力下做到？”

\subsection{误区二：用输赢评价自己打得好坏}

新手最容易被短期结果牵着走。赢了几手，就觉得自己状态很好；输了一手大底池，就觉得自己被针对或者运气太差。

但在德州扑克中，短期结果具有很大波动性。正确打法可能输，错误打法也可能赢。如果只用输赢评价自己，就会形成错误反馈。

比如，你用很差的底池赔率追听花，河牌真的中了，于是你觉得这次跟注很成功。但从长期看，如果你的跟注价格不合理，这类决策会持续亏损。相反，你用强牌在翻前全下，被对手小对子击中暗三条赢走底池，这不代表你的全下错误，只是短期结果不利。

学习扑克必须把注意力从“这手赢没赢”转向“这手是否打对”。

\subsection{误区三：认为心态只是输多了才需要}

很多人以为，心态问题只发生在连续输钱之后。实际上，赢钱也会让人失控。

输钱后的失控通常表现为追损、报复、过度跟注；赢钱后的失控则可能表现为轻敌、放松纪律、玩更多边缘牌、过度自信。无论是过度沮丧还是过度兴奋，都会破坏决策稳定性。

优秀牌手的心态不是永远平静，而是不管短期结果如何，都尽量让每一个行动回到决策框架中。赢牌不膨胀，输牌不崩溃，才是真正稳定的状态。

\subsection{误区四：把多打牌当成唯一训练方式}

如果一个人每天都用错误方式打牌，他不是在训练，而是在强化坏习惯。

很多娱乐玩家打了很多年，水平却没有明显提升，原因就在于他们缺少复盘和修正机制。他们记得很多精彩牌局，却很少系统分析自己的长期漏洞；他们记得被河杀的痛苦，却不记得自己翻前入池是否合理；他们抱怨对手运气好，却不检查自己是否给了对手合适的价格。

真正有效的训练，需要“打牌—记录—复盘—修正—再打牌”的闭环。没有复盘，多打只是增加样本；有复盘，多打才会转化为经验。

\subsection{误区五：忽视资金管理}

初学者常常觉得资金管理是高级玩家才需要考虑的事情。实际上，越是新手，越需要资金管理。因为新手技术尚不稳定，情绪控制能力也较弱，如果再没有资金边界，很容易在短期波动中做出失控行为。

资金管理的意义不仅是保护本金，更是保护心态。当你坐在一个超出自己承受能力的牌局中，每一次下注都会让你紧张，每一次被加注都会让你害怕，每一次输牌都会影响生活。这样的状态下，很难做出高质量决策。

一名想长期进步的牌手，必须先学会保护自己的训练环境。

\section{实战思考框架}

本章的核心输出，是一套适用于新手成长阶段的训练框架：

\begin{center}
\textbf{稳定状态 $\times$ 正确框架 $\times$ 持续复盘 $\times$ 长期执行}
\end{center}

这不是一句口号，而是判断自己是否走在正确学习路径上的标准。

\subsection{稳定状态：先确保自己能正常决策}

打牌前，首先要检查状态。状态不好时，不是不能打，而是要降低预期、减少时长、控制风险，甚至直接停止。

可以用以下问题检查自己：

\begin{itemize}
\item 我现在是否疲劳？
\item 我是否因为上一局输赢影响情绪？
\item 我是否有追损冲动？
\item 我是否开始觉得“随便玩玩也没关系”？
\item 我是否还能认真观察位置、范围、下注尺度和对手倾向？
\end{itemize}

如果这些问题中有多个答案不理想，说明当前状态已经不适合做高质量决策。

\subsection{正确框架：每一手牌都要有思考步骤}

新手最怕大脑空白。解决方法不是记住所有复杂理论，而是先建立简单的思考顺序。

在本课程中，一手牌的基础思考路径可以概括为：

\begin{enumerate}
\item 翻前：我的位置和手牌是否支持入池？
\item 翻牌：这个牌面对谁更有利？我的牌属于什么类型？
\item 转牌：这张牌改变了什么？我是否应该继续进攻或控制底池？
\item 河牌：我的行动是价值、诈唬、摊牌还是弃牌？
\item 复盘：如果不知道结果，我的决策是否仍然合理？
\end{enumerate}

这个框架并不能立刻解决所有复杂局面，但它能帮助新手从混乱直觉中走出来，开始按步骤思考。

\subsection{持续复盘：让经验变成能力}

复盘不是为了责备自己，而是为了找到下一次可以改进的地方。

建议新手每次打牌后至少记录三类手牌：

\begin{itemize}
\item 大底池手牌：因为它们对结果影响最大；
\item 困难决策手牌：因为它们暴露了认知盲区；
\item 情绪波动手牌：因为它们最容易产生非理性行动。
\end{itemize}

每手牌复盘时，可以按照以下模板记录：

\begin{enumerate}
\item 有效筹码是多少？
\item 我在什么位置？
\item 翻前行动是什么？
\item 公共牌如何变化？
\item 每条街下注尺度是多少？
\item 当时我是怎么想的？
\item 我的行动目的是什么？
\item 如果不看结果，这个决策是否合理？
\item 下次遇到类似局面，我要如何调整？
\end{enumerate}

复盘越具体，进步越快。只写“这手运气不好”没有意义；写清楚自己在哪个节点失去纪律，才有改进价值。

\subsection{长期执行：接受波动，重复正确决策}

德州扑克最难的地方之一，是正确决策并不总是立刻得到奖励。你可能连续几次做对却输钱，也可能偶然做错却赢钱。短期反馈经常会误导人。

长期执行意味着，你要允许自己在正确决策下承受短期亏损，也要警惕自己在错误决策下偶然获利。真正优秀的牌手，不是靠一次精彩读牌证明自己，而是靠长期稳定执行积累优势。

新手阶段最重要的不是追求复杂操作，而是坚持几个简单原则：

\begin{itemize}
\item 不在差状态下长时间打牌；
\item 不用过宽范围随意入池；
\item 不因为不甘心而跟注；
\item 不为了试探而下注；
\item 不用短期输赢评价全部决策；
\item 每次打完至少复盘几手关键牌。
\end{itemize}

这些原则看似简单，但能长期做到的人并不多。而扑克的差距，往往就来自这些基本原则的长期执行。

\section{牌例拆解}

本章不是具体技术章，但我们仍然通过两个简化牌例，说明状态、心态、EV 思维和复盘习惯如何影响实战决策。

\subsection{牌例一：强牌输掉后，下一手牌该如何处理}

\subsubsection*{牌局背景}

有效筹码 100BB。你在上一手牌中用 \texttt{AA} 翻前 3-bet，对手用一手较弱的同花连张跟注。翻牌你持续下注，转牌继续下注，河牌对手完成顺子并赢下大底池。你感到非常不满，认为对手不应该用这样的牌跟到最后。

下一手牌，你在中位拿到 \texttt{KJo}。前面一位玩家加注到 3BB，你还没有完全从上一手的情绪中恢复。

\subsubsection*{新手常见想法}

很多新手此时会想：

\begin{itemize}
\item “上一手太倒霉了，这手我要打回来。”
\item “\texttt{KJ} 也不差，可以看看。”
\item “这个人刚才运气那么好，我不能一直让他。”
\item “如果翻牌中到 K 或 J，也许能赢回来。”
\end{itemize}

这些想法看似在分析手牌，实际上大部分是情绪驱动。你并不是因为 \texttt{KJo} 在这个位置面对前位加注具有足够价值才继续，而是因为上一手牌的结果影响了当前判断。

\subsubsection*{理性分析}

面对前面玩家加注，中位的 \texttt{KJo} 通常是一手容易被压制的牌。它看起来有高牌价值，但经常被更强的 \texttt{AK}、\texttt{KQ}、\texttt{AJ}、\texttt{AQ} 控制。即使翻牌击中一对，也可能陷入踢脚较弱、难以打大底池的局面。

更重要的是，你当前的状态并不稳定。刚刚输掉大底池后，你有明显追损倾向。此时继续用边缘牌入池，很容易在翻后做出更多被情绪影响的决策。

\subsubsection*{推荐行动}

弃牌。

这次弃牌不是因为 \texttt{KJo} 永远不能玩，而是因为在当前局面中，位置、前序行动、手牌质量和个人情绪状态都不支持继续。优秀牌手不会因为上一手牌的结果改变下一手牌的标准。

\subsubsection*{本手牌教训}

扑克中最危险的错误，常常不是发生在输掉那手牌本身，而是发生在输牌之后的几手牌。强牌被反超不可避免，但因为情绪失控继续扩大损失，是可以避免的。

优秀牌手要学会把每一手牌重新归零。上一手牌已经结束，下一手牌仍然必须按照位置、范围、牌力和状态做决策。

\subsection{牌例二：赢了底池，不代表打得正确}

\subsubsection*{牌局背景}

有效筹码 100BB。盲注 1/2。你在按钮位拿到 \texttt{Q8s}。前面两位玩家跟注，你也跟注，小盲弃牌，大盲过牌。

翻牌为 \texttt{K72}，其中两张与你同花。大盲下注 6BB，一位玩家跟注，另一位玩家弃牌。你持有同花听牌，选择跟注。

转牌为 \texttt{3}，没有完成同花。大盲下注 18BB，前面玩家弃牌。你觉得“如果河牌来同花就能赢大底池”，于是继续跟注。

河牌发出同花，你完成同花。对手下注，你加注，对手跟注，你赢下一个大底池。

\subsubsection*{新手常见想法}

很多新手会认为：

\begin{itemize}
\item “我听花跟注是对的，因为最后中了。”
\item “有同花听牌就应该坚持。”
\item “赢了大底池，说明这手打得不错。”
\end{itemize}

这正是结果导向的典型表现。河牌中了同花，只说明这一次结果很好，并不能自动证明前面每一次跟注都是正确的。

\subsubsection*{理性分析}

听牌能不能跟注，关键不在于“有没有机会中”，而在于价格是否合适、隐含赔率是否足够、对手范围是否会在你击中后继续支付。

翻牌面对下注和一位跟注，你持有同花听牌，跟注通常有一定合理性，因为你有直接权益，也有位置优势。

但转牌情况发生了变化。对手继续下注较大尺度，说明他的范围可能更强。此时你只有一张牌可以看，完成同花的概率有限。如果对手下注尺度过大，而你没有足够的直接赔率或隐含赔率，继续跟注就可能是亏损决策。即使这一次河牌真的完成同花，也不能说明转牌跟注一定正确。

\subsubsection*{推荐复盘方式}

这手牌不能简单写成“听花成功，赢了大底池”。正确复盘应该记录：

\begin{itemize}
\item 翻牌跟注的底池赔率是否合理？
\item 转牌面对大注时，我需要多少胜率？
\item 我的同花完成后，对手是否真的会继续支付？
\item 如果河牌没有来同花，我是否会直接损失转牌跟注金额？
\item 长期重复这个转牌跟注，是否盈利？
\end{itemize}

\subsubsection*{本手牌教训}

赢牌不等于打得好，输钱也不等于打得差。扑克复盘必须回到决策过程，而不是只看最终结果。

新手尤其要警惕那些“赢了但打错”的手牌，因为它们最容易强化坏习惯。一次幸运的结果，可能会让你在未来重复很多次亏损决策。

\section{本章总结}

本章讨论的是成为优秀牌手的底层基础，而不是某一个具体技术点。初学者要尽早明白，德州扑克不是简单的比牌大小，也不是靠感觉猜对手有没有牌。它是一项在不完整信息下持续做决策的竞技运动。

优秀牌手的成长，离不开四个要素：

\begin{center}
\textbf{稳定状态 $\times$ 正确框架 $\times$ 持续复盘 $\times$ 长期执行}
\end{center}

稳定状态是技术发挥的前提。疲劳、急躁、追损、报复心理和过度自信，都会让牌手偏离正确策略。

正确框架是学习效率的保障。没有框架的多打牌，只是在重复旧习惯；有框架的训练，才能把每一手牌转化为可分析、可修正的经验。

持续复盘是进步的关键。复盘不是为了证明自己运气差，也不是为了责备自己，而是为了找出决策过程中的漏洞。

长期执行是牌手成熟的标志。短期输赢不可控，但长期决策质量可以控制。真正优秀的牌手，不会因为一手牌的输赢改变自己的原则。

从下一章开始，我们将进入一手牌的第一个技术环节：翻前。翻前是整手牌的地基。很多翻后困难，其实都来自翻前入池范围过宽、位置意识不足和被动跟注过多。只有先把翻前打扎实，后续每条街的决策才会变得清晰。

\section{课后训练}

\subsection*{训练一：状态检查记录}

下一次打牌前，请用 1 到 5 分给自己的状态打分：

\begin{itemize}
\item 精力状态：\underline{\hspace{2cm}} 分
\item 情绪稳定度：\underline{\hspace{2cm}} 分
\item 专注程度：\underline{\hspace{2cm}} 分
\item 是否有追损或急于赢钱的想法：\underline{\hspace{2cm}} 分
\item 是否愿意严格执行翻前纪律：\underline{\hspace{2cm}} 分
\end{itemize}

如果多项评分低于 3 分，建议减少游戏时长，降低级别，或者直接停止本次训练。

\subsection*{训练二：记录三手关键牌}

每次打牌后，记录至少三手关键牌：

\begin{enumerate}
\item 一手大底池牌；
\item 一手你当时不知道怎么决策的牌；
\item 一手你明显受到情绪影响的牌。
\end{enumerate}

记录时至少包括：位置、有效筹码、翻前行动、公共牌、下注尺度、你的行动、你当时的想法和最终结果。

\subsection*{训练三：区分结果与决策}

从最近打过的牌中，找出两手牌：

\begin{itemize}
\item 一手你赢了但可能打得不好的牌；
\item 一手你输了但可能打得正确的牌。
\end{itemize}

分别回答：

\begin{enumerate}
\item 如果不知道结果，我当时的决策是否合理？
\item 我的行动有没有明确目的？
\item 我是否受到了情绪影响？
\item 如果下一次遇到类似局面，我会如何调整？
\end{enumerate}

\subsection*{训练四：建立停止条件}

请为自己写下三条停止打牌的条件，例如：

\begin{itemize}
\item 当我连续输掉两个大底池并明显想追损时，停止；
\item 当我开始用平时不会玩的边缘牌入池时，停止；
\item 当我感到疲劳、注意力下降或不再认真观察对手时，停止。
\end{itemize}

停止条件必须在打牌前制定，而不是在情绪失控后临时决定。

\subsection*{训练五：本章思考题}

请用自己的话回答以下问题：

\begin{enumerate}
\item 为什么德州扑克不能只靠感觉打牌？
\item 为什么短期输赢不能直接证明决策好坏？
\item 情绪失控通常会通过哪些方式影响决策？
\item 为什么资金管理也是牌手能力的一部分？
\item “稳定状态 $\times$ 正确框架 $\times$ 持续复盘 $\times$ 长期执行”中，你目前最薄弱的是哪一项？
\end{enumerate}
